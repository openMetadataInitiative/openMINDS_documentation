###############
Getting started
###############

This chapter introduces the basic concepts required to work with the openMINDS metadata framework.

It provides:  

- a brief introduction to JSON-LD as a format for linked metadata  
- a conceptual overview of how metadata is structured and represented in openMINDS 

If you are already familiar with Linked Data, JSON-LD, and the basic concepts of openMINDS, you can skip this chapter and proceed directly to `Working with openMINDS`_.

.. note::

   The examples illustrate general principles and are based on the latest openMINDS version. In other versions, schema names, properties, and requirements may differ and require adaptation.
